# AI Usage Report

## AI Tools Used
The AI tools used in this project include ChatGPT, GitHub Copilot, and Google Gemini.

## How AI Was Used
AI was used to assist in generating Flutter UI scaffolding, QR code scanner integration, and general Firebase Hosting configuration guidance. AI was also used to help debug code errors and suggest improvements during development.

## What I Implemented or Modified
I reviewed and modified the generated code to match the project requirements. The application logic, navigation flow, form inputs, and SQLite data storage were implemented and adjusted manually to ensure the system works correctly across supported platforms.
